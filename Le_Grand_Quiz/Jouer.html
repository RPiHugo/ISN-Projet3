Le Grand Quiz

Vous avez le choix entre quatre catégories de questions pour notre jeu, veuillez cliquer sur la catégorie à laquelle vous souhaitez jouer ci-dessous :
  • Code de la route
  • Culture numérique
  • Le monde du sucre dans la culture
  • Les vins

Règles du jeu :

Pour commencer, choisissez votre mode de jeu. Le mode Solo permet de s'entrainer avec les questions du thème. Il n'y a pas de limite, cliquer sur Retour si vous souhaiter retourner sur le menu du jeu. Le mode Multi est le même jeu mais pouvant confronter 4 joueurs en même temps. Une partie se joue en 5 questions. Les scores des 4 joueurs sont ensuite révélés. Pour quitter ce mode veuillez cliquer sur le bouton Retour qui permettra de libérer votre place sur le serveur. Si jamais les joueurs précédents n'ont pas cliqué sur le bouton Retour, veuillez, sur l'écran d'accueil, cliquer sur le bouton Réinitialiser le serveur pour recommencer une nouvelle partie.
